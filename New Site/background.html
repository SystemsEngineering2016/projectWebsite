Project Background

An overview of the beginings of our project

Our Problem Statement

Our aim is to develop an application that can act as a data dictionary editor and data mapping tool. The application consists of interfaces and a relational data model that is implemented on a relational database. The application accepts data dictionaries and systems as its data

Specific Challenges

  • Develop a good understanding of databases, data dictionaries and other topics that will help with the project
  • Carry out research to identify the most suitable tools and frameworks for developing the application
  • Develop an application that should act as a data dictionary editor and data mapping tool
  • Develop an application that should at least satisfy all the ‘must have’ and ‘should have’ requirements
  • Develop an application that should be user friendly

Our Clients


Across Health

Across Health provide digital innovation in healthcare. They work with healthcare providers, clinicians, hospitals trusts and research partners to develop modern digital technology that will help improve the quality of patient care. They achieve this through “design, development, implementation and support of open source software solutions”. They have been appointed OpenEyes Foundation implementation, integration and support provider.

Across Health Website: http://www.acrosshealth.co.uk/new-about-us/

OpenEyes™

OpenEyes™ helps to provide Electronic Medical Record (EMR) applications for ophthalmology (eye care) which can be used by clinicians to access information about their patients. It is an open source project that has been developed over time using input from hospitals, institutions, academic departments, companies and individuals. OpenEyes™ is a cost effective option for health care providers as it has no license fees and cannot be sold to 3rd parties.

OpenEyes provides allows efficient input of data , analysis and interpretation of results. The application combines data from sources such as imaging machines to meet agreed national and international data sets. The data sets collected provide a representation for the different areas of each ophthalmic specialty. This can aid in “tracking clinical outcomes, research and development and clinical audit”.

OpenEyes website: http://www.openeyes.org.uk
