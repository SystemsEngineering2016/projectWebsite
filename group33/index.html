OpenEyes and Across Health: A Data Dictionary Editor

Our 2016 Systems Engineering Project, in association with OpenEyes and AccrossHealth

  • Learn More

Introducing Maple, the all in one data dictionary editor
and data mapping tool

Our client has a database system which is fairly complicated and not thoroughly documented. Our goal is to provide software which allows them to construct documentation for specific versions of their system, then provide mapping between separate versions and other systems.

Key Features

Maple will provide the ability to add and edit data dictionaries

The user will be able to define and edit data points for a data dictionary that they are editing

Maple provides the ability to add and edit systems for data dictionaries

The user will easily be able to create mappings between different data points

The system will provide the ability to handle system versioning

The system will have an easy to use user inteface in order to carry out required tasks
