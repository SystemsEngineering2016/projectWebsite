Our Prototype

We designed some non-functioning prototypes for our user interface which are shown below. We will use these to produce functioning prototypes which can be seen on the Prototypes page.

In order to gain a better understanding of what our project should look like, we created some functional User Interface prototypes. We used the paper prototypes in our UI page in order to come up with these prototypes. They give us a visual undertanding on how components of the website connect with each other and the overall architecture of the site.

In order to create the prototye we used Bubble as it is a quick and efficient way to create a functioning prototype. It allows multiple users to edit the prorotype at once meaning the whole team could colllaborate at the same time when necessary

The prototype can be seen at https://bubble.is/site/maple/
