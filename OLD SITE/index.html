THE PROJECT

The Project

Our initial problem statement

Our aim is to develop an application that can act as a data dictionary editor and data mapping tool. The application consists of interfaces and a relational data model that is implemented on a relational database. The application accepts data dictionaries and systems as its data

Key Features

Project Key Features

Our project will have the following key features:


1) Provides the ability to add and edit data dictionaries

2) Provides the ability to define data models for a data dictionary

3) Provides the ability to extract data and load data models

4) Provides the ability to map data

5) Provides the ability to handle system versioning
